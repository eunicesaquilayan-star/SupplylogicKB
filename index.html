DemandBridge Function

HOW TO LOG IN

1. Once the log in page load type your log in ID and hit enter *note if you type anyting it's not visible on your end but system capture.

2. If ID is correct, the option to put your password will show, again it's not visible when you type just hit enter.

3. Once log in successfully wait for the next prompt and choose number 2 (DB GUI).

4. Wait for the prompt and then click accept

5. Another prompt will show, please click continue.

6. Folders will show, please double click DB Distributor.

7. Then under Distributor click Accounts Receivable.

8. Then choose setp up.

9. Then Customer folder.

HOW TO SEARCH ORDER

1. Click ORDER LOOK UP tab

2. Type the order number, then click Order history by item.

3. Once this prompt show click proceed.

4. This screen will show close and open orders, Orders that are highlighted in green are still open. Double click the order number

5. Then it will route you to the order details
